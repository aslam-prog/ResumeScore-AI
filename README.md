✨ Features
Feature	Description
⚡ Resume Score	Full ATS score out of 100 — keywords, impact verbs, structure, readability & achievements
🎯 JD Matcher	Paste any job description → get keyword match %, hits & missing keywords instantly
📊 Industry Benchmark	Compare your score against 12 role-specific industry averages with percentile ranking
🎙️ Tone Analyzer	Detect passive/weak phrases and get stronger, assertive alternatives side by side
💼 LinkedIn Headlines	Generate 5 ranked LinkedIn headlines with one-click copy
✍️ Summary Rewriter	3 rewritten summary versions — Executive, Technical & Balanced tones
🌙 Dark / Light Mode	Sleek dark mode by default, toggle anytime — preference saved in browser
💡 Resume Tip of the Day	Rotating career tips to improve your resume instantly
📸 Export Score Card	Export your resume score as a beautiful shareable card

🛠️ Tech Stack
Vanilla HTML / CSS / JavaScript — zero dependencies, zero build tools
Pure rule-based engine — no AI API, no backend, fully offline capable
localStorage — saves theme preference & resume scored counter
Google Fonts (Satoshi) — clean modern typography via CDN
Single file — entire app is one `index.html` file, ~66KB
---
📦 Getting Started
```bash
# Clone the repo
git clone https://github.com/aslam-prog/ResumeScore-AI.git
cd ResumeScore-AI

# Just open the file!
open index.html
```
That's it. No `npm install`. No backend. No API keys. No internet required after first load.
